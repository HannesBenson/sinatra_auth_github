    Sinatra Auth GitHub

    We Sinatra at GitHub for building small applications that do something well. A lot of times we want to enable those apps to allow GitHubbers to login and see what we've built. We use organizations and GitHub OAuth](http://developer.github.com/v3/oauth/) to make this happen. This way we can ensure that our coworkers can see the data, but other random GitHub users.

    Features

    • authenticate a user against GitHub's OAuth service
    • provide an easy way to make API requests for the authenticated user
    • optionally restrict users to a specific GitHub organization
    • optionally restrict users to a specific GitHub team
